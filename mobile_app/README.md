### Prerequisites
- Flutter SDK (latest version)
- Android SDK (for Android development)
- Xcode (for iOS development on Mac)

### Installation

1. Get Flutter dependencies:
```bash
flutter pub get
```

2. Generate Hive adapters:
```bash
flutter pub run build_runner build
```

3. Run The App
``bash
# For Android
flutter run

# For iOS
flutter run -d ios
```
